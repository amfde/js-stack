Usando o MongoDB

Usando MongoDB

O MongoDB chegou com um forte apelo por ser um banco de fácil uso e de alta performance e disponibilidade. Já conhecemos algumas de suas virtudes quando falamos dos PRIMERS desse Stack.

Nesse capítulo vou te mostrar como instalar e usar o MongoDB localmente para apoiar o desenvolvimento de nossas aplicações.

Vale resaltar, como você já sabe. Que um dos grandes benefício do MongoDB nesse Stack é que um banco orientado-a-documentos facilita muito o desenvolvimento já que podemos trabalhar com objetos JSON trafegando por todo o nosso Stack. Trafegando o mesmo tipo de dados desde o banco, MongoDB, passando pelo servidor em Node até o frontend em React.

Então, vamos a intalação e uso do MongoDB...
